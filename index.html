About the port

What is this?

OneShot: World Machine Edition is the expanded re-release of the cult RPG by FutureCat, featuring Niko, a child tasked with restoring the sun to a dying world — a world that knows you exist.

This is an unofficial fan port that brings the full World Machine Edition experience to Android, including the Solstice route, the Barrens, and all WME-exclusive areas — touch-optimised, no emulator needed.

OneShot and all related content belong to FutureCat. This port is non-commercial and made out of love.

Setup tool

ResAdder

⚙ ResAdder v1.0

ResAdder automatically adds your game resources to the APK — no manual file copying required. Place your game files and the APK in the same folder as the extracted ResAdder zip, run it, and it handles the rest.

You'll need a Windows, Linux, or macOS build of OneShot WME. Specifically, the content and gamedata folders from your copy of the game, plus the downloaded APK, must all sit in the same directory as the extracted ResAdder tool.

Note: The Windows build of ResAdder is available now. Linux and macOS builds are coming soon.

Get the game

Download

Download the APK and side-load it on your Android device. You'll need to enable Install from unknown sources in your device settings.

Requires Android 6.0 (Marshmallow) or later. Tested on phones and tablets. A file manager or adb install also works.

You must own a legitimate copy of OneShot or OneShot: World Machine Edition. Use ResAdder to automatically inject your game's content and gamedata folders into the APK before installing.
